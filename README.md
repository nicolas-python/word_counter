# Word Counter

Ein Programm, das die Anzahl der Wörter ,Buchstaben , Zeichen und Sonderzeichen in einem Text zählt.

## Funktionen

- zählt Wörter
- zählt Buchstaben 
- zählt Sonderzeichen 
- zählt Zahlen

## Nutzung 

Programm starten mit : 

python main.py

Danach einen Text eingeben der analysiert werden soll und Enter drücken.

## Beispiel 

Eingabe 
Hallo ich bin Nico!234

Sie haben Buchstaben: 15
Sie haben Woerter: 4
Sie haben Zahlen: 3
Sie haben Sonderzeichen: 1
